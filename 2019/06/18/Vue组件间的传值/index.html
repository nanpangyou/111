Vue 组件间的传值

Vue 组件间的传值，说白了就是父子组件和兄弟组件之间的传值。
这次梳理一下父子组件之间的传值。

Vue 父子组件之间的传值 👨👦

传值主要涉及两种类型的传值

  • 单纯的值
  • 函数（方法）

单纯的值

单纯的值传递，主要是想将父组件data对象中的值传入子组件，在子组件中使用。

主要使用 props

函数传递

父组件的函数传递给子组件，一般用在子组件内部更新后让父组件刷新状态。

主要使用 this.\$emit()

在父组件中直接获取子组件的数值或者函数

在 Vue 中，可以使用 ref 来做到类似上古时代使用 id 绑定元素的功能
