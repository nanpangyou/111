使用karma-mocha-sinon和travis尝试做持续测试

发表于 2019-12-04
  1. Karma（[ˈkɑrmə] 卡玛）是一个测试运行器，它可以呼起浏览器，加载测试脚本，然后运行测试用例
  2. Mocha（[ˈmoʊkə] 摩卡）是一个单元测试框架/库，它可以用来写测试用例
  3. Sinon（西农）是一个 spy / stub / mock 库，用以辅助测试
  4. travis 是一个免费的 CI 平台
